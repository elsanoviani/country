# Project Vue Vite

Deskripsi singkat: Aplikasi frontend Country menggunakan Vue 3 dengan Vite.

## Teknologi
- Vue 3
- Vite
- Vue Router
- Axios / Fetch API

## Instalasi
1. Clone repository:
```bash
git clone https://github.com/username/vue-vite-project.git
cd vue-vite-project
```
2. Install dependencies:
```bash
npm install
# atau pakai yarn
yarn install

```
2. Menjalankan project:
```bash
npm run dev
# atau yarn dev
```
